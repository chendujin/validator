# validator

### 判断是否为合法的用户名
is_username($username, $min_len, $max_len, $mode = 'mix')

### 判断是否为合法的密码
is_password($password, $min_len, $max_len, $mode = 'mix')

### 判断是否为QQ号码
is_qq($qq)

### 判断是否为手机号码
is_mobilephone($mobilephone)

### 判断email格式是否正确
is_email($email)

### 验证身份证号码是否合法
is_idcard($vStr)
